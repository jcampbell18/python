# Hangman Game in Python

- Simple Game
- Guess letters one at a time to solve the word puzzle.
    - The computer selects a word from a list
    - User types in a letter
    - 6 guesses
    - Displays ASCII art

## Coding

    - Focuses on:
        - Lists
        - Conditionals
        - Randomization
        - For Loops
        - While Loops
        - Importing Modules
